# Especificação Estrutural: Home Page

## 1. Header (Navegação)
* Logo à esquerda.
* Links: Início, Imóveis, Sobre, Contato.
* CTA Secundário: "Falar no WhatsApp".

## 2. Hero Section (Dobra Principal)
* **Objetivo:** Impacto visual e busca rápida.
* **Layout:** Imagem de fundo de alta qualidade (imóvel de luxo) ou carrossel sutil.
* **Elementos:** Headline de autoridade (ex: "Imóveis Exclusivos em [Região]").
* **Componente Chave:** Barra de busca rápida (Dropdown "Cidade" + Dropdown "Tipo" + Botão "Buscar").

## 3. Featured Properties (Destaques)
* **Objetivo:** Mostrar o "filé mignon" do portfólio.
* **Layout:** Grid de 3 colunas (Desktop) / 1 coluna (Mobile).
* **Card do Imóvel:** Foto principal, Título, Bairro/Cidade, Preço, ícones de quartos/área.

## 4. Why Us (Autoridade)
* **Objetivo:** Diferenciar do portal genérico.
* **Layout:** Ícones + Texto curto.
* **Conteúdo:** "Curadoria Especializada", "Atendimento Personalizado", "Segurança Jurídica".

## 5. Social Proof / Sold Properties (Prova Social)
* **Objetivo:** Mostrar que você vende de fato.
* **Layout:** Grid menor ou slider.
* **Visual:** Cards de imóveis com tag "VENDIDO" em destaque.

## 6. Footer
* Links rápidos.
* Dados de contato (CRECI, Endereço, Telefone).
* Copyright.
